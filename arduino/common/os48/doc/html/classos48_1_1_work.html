void os48::Work::cancel ( )

Cancels the work. Releases all waiting task which are joined to this work.

  • When the state is WkStQueuing, the work instance is dequeued.
  • When the state is WkStRunning, you have to check in your function implementation if the cancellation token became true with Work::isCancellationRequested and return from the function. The state is already changed.
  • When the state is something else, nothing happens.
bool os48::Work::join ( )

The task calling this method will be blocked as long as the work has not been completed or canceled.

Returns
false if the work instance has not been linked with a Sync instance. Take a look to the constructor Work::Work().
