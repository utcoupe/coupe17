Detailed Description

Contains primitive functions to add task in a waiting queue and release them. You can use this class as an event synchronization object.

bool os48::Sync::releaseOne ( )

Releases a task pending in the waiting queue then yield if a task with a higher priority is ready.

Returns
false if there is no waiting task, true otherwise.
Note
Depending of the release mode you choosed with SyncReleaseMode, the released task can be the oldest waiting task or the oldest waiting task with the highest priority.
