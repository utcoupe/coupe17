bool os48::Semaphore::tryAcquire ( )

The current task tries to acquire the semaphore.

Returns
false if the maximum of task having acquired the semaphore is already reached. true if the semaphore is acquired.

Example:

if (sem.tryAcquire())
{
[CRITICAL SECTION]
}
else
{
[DO SOMETHING ELSE]
}
void os48::Semaphore::acquire ( )

The current task tries to acquire the semaphore.

If the maximum of task having acquired the semaphore is already reached, then the task will be added in the waiting queue.

void os48::Semaphore::release ( )

Releases the semaphore in order to let another task acquire it.

Note
Yields the current task to another if a task with a higher priority is ready.
