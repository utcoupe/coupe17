Detailed Description

template<typename T>
class os48::QueueItem< T >

QueueItem can be used to create queues.

Template Parameters
TThe type of the data pointed by the queue item.
Note
You can also extend this class to your own class. The data pointed by the queue item will be your class instance.
class MyClass : QueueItem<MyClass>
