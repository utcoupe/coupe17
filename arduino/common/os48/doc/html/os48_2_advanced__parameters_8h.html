#define OS48_SCH_RROBIN_BASE_TIME
#define OS48_KERNEL_TICK_OCR0A

OCR0A value used to perform one kernel tick.

#define OS48_IDLE_TASK_TIME_SLICE

Kernel ticks applied before performing the context switch if the current task is the idle task.

#define OS48_CHECK_SLEEPING_TASKS_AT_EACH_TICK

If you set 1, the kernel checks at each kernel tick if a sleeping task has to be waked up (this parameter could decrease resources). Otherwise the check will be performed before the context switch, so less often.
