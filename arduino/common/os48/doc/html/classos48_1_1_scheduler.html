size_t  stackSize,
TaskPriority  priority = PrNormal,
bool  autoStart = true 
)

Creates a task.

Parameters
fncA void function without params associated to the task.
stackSizeThe size of the stack, make sure to specify an enough quantity of memory to avoid stack overflows or indefinite behaviours in your program.
priorityThe priority of the task os48::TaskPriority.
autoStart'false' indicates that the task can be queued to be later started.
Returns
A pointer to the newly task or NULL if an error occured.
Note
You can't delete a task with the delete or free keyword, use the Scheduler::deleteTask() method instead.
Error codes
Scheduler::getLastError() can return the following codes:
  • SchErrArgs : If the priority value is out of bounds or if fnc is NULL.
  • SchErrAlloc : There is not enough free memory to create the task.
  • SchErrTaskCorrupted : The constructor of the task has failed to initialize task parameters or the stack.

bool os48::Scheduler::yieldTo ( Task *  task,
bool  delayed = false 
)

Yields the current task to another task (works only in a cooperative scheduling algorithm).

Parameters
taskThe task to execute next.
delayedYields the current task only at the next Scheduler::yield() call. This can also be done by calling a kernel function like Task::suspend() or Semaphore::acquire(). A second call to Scheduler::yieldTo() with delayed = true cancels the first call.
Returns
false if an error occured.
Error codes
Scheduler::getLastError() can return the following codes:
  • SchErrWrongSchedulingPolicy : If the Scheduling policy is not SchPolicyCoop.
  • SchErrTaskIncorrectState : If the task is not alive.
See also
Task::isAlive()
Note
Use this function preferrably for the cooperative scheduling algorithm (SchPolicyCoop).
