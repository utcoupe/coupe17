Detailed Description

template<typename T, uint8_t N>
class os48::MemoryPool< T, N >

Creates Memory pool for fixed size data.

Template Parameters
TThe type of the data pointed by the queue item.
NThe numbers of block of T-size to reserve. N shall be <= 255 and > 0. The memory pool instances have as main advantage to allocate or deallocate the data in O(1). As compensation, the reserved memory space is not available for allocate other sizes of data.
Warning
This class does not provide any check on index passed as arg to functions.
